
#### Installation
```bash
# Install node modules - this includes those for production and development
# You only need to do this once :)
npm install
```

#### Development

```bash
# Starts the server with Hot Reloading
# Run webpack through
npm run dev

```

#### Use

You can test the API at `http://localhost:3001`
